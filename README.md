# FE (Fundamental Information Technology Engineer Examination)
基本情報技術者試験 기본정보기술자시험

본 레지토리는 일본 국가자격증 기본정보기술자의 공략정리본 입니다.

## 출제 범위
오전 시험은 기술계 / 관리 / 전략 으로 나뉘어지며 총 80 문제 중 48문제(60%)를 합격으로 한다.
